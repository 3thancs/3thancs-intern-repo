**Which terminal client did you choose? Why?**
* The terminal client I have chosen is the **PowerShell 7 (Core) and Windows Terminal**. 
* Windows terminal enables me to **multitask**, where I have a Python script on one tab, and a C++ script for my HIN data structure project in another tab.
* The terminal provides **visual indicators** which states the Git branch, commits and current directory - without the need to run inefficient commands such as pwd or git stats
* Enables the **UI customization** of PowerShell 


**What customizations (if any) did you make?**
* **Prompt Engine**
  * **The Oh My Posh prompt engine** replace the standard PowerShell prompt. It provides a tracking status on the current repository, branch and execution of the command

* **Icons and typography** 
  * I installed and configured the font face appearance with **Caskaydia Cove Nerd Font**. This enables the visual icons for Git repositories and folders to be rendered with visual indicators
  
* **UI amendments**
  * **Background opacity** was adjusted to 70 percent, giving it a translucent display


**What was the most useful command you learned today?**
   * Performing the **setup** using the command 'New-Item -Path $PROFILE -Type File -Force'. This command resolved the issue 'file path not found' error. I used 'Force' to create the notepad file and directory structure and file to set up the PowerShell profile.
